======================
Glossary
======================

.. glossary::

    Interaction model
        Defines the way user's interact with an Alexa Skill. :term:`Intents` define what types of
        requests you expect to receive from your users with or without a specific set of terms.
        :term:`Sample utterances` map natural language phrases to an Intent. See Amazon docs on
        `interaction models`_.

    Intent
    Intents
        A pre-defined intention which the Alexa platform determines after listening to and parsing
        a user's speech

    Sample utterances
        A pre-defined list of natural language phrases which map to a specific Intent



.. _interaction models: https://developer.amazon.com/public/solutions/alexa/alexa-skills-kit/docs/alexa-skills-kit-interaction-model-reference
